Case Study

Problem

Retail teams need a clear view of revenue, margin, and returns across regions and categories.

Approach

Using a public retail dataset, I cleaned data in Power Query, modeled a star schema, and built DAX measures.

Data Model

Star schema with fact tables for sales and returns plus product, customer, store, and date dimensions.

Dashboard Highlights

Executive KPIs, category and region drill-downs, and customer segmentation views.

Automation

Optional Power Automate alerts for margin drops and return spikes.

Outcome

Creates a single view of performance with clear drivers and actionable drill-downs.

Improvements

Next steps include publishing to Power BI Service, adding RLS, and documenting a data dictionary.
